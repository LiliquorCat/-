# 🎮 海拉的钱包 - 无期迷途抽卡资源计算器

## 📖 项目简介

**海拉的钱包** 是专为无期迷途设计的抽卡资源计算器，帮助局长科学规划抽卡资源，避免资源浪费。该工具可以近乎全面统计局长当前拥有的抽卡资源，并预估未来可获得的各类资源，让局长能够合理规划抽卡策略。<br>
https://liliquorcat.github.io/-/

## ✨ 功能特点

### 🎯 核心功能
- 💎 **当前资源管理** - 记录异方晶、璀璨晶源、搜索令等现有资源
- 📈 **日常积累计算** - 自动计算日常、周常、月常奖励
- 🔮 **潜在资源统计** - 统计主线关卡、角色养成等潜在资源
- 🎯 **活动资源估算** - 估算活动可获得抽卡资源
- 💰 **氪金资源规划** - 规划月卡、礼包等付费内容
- 📊 **可视化分析** - 通过饼状图直观展示各模块资源占比

### 🛠️ 特色功能
- 🔄 **自动数据保存** - 所有数据自动保存在本地浏览器中
- 📱 **响应式设计** - 完美适配桌面端和移动端
- 🌙 **日夜模式切换** - 提供舒适的视觉体验
- 🗂️ **详细资源分类** - 7大模块全面覆盖各类资源来源

## 📋 使用指南

### 基本操作步骤：
1. 在 **「当前资源」** 模块输入你现有的抽卡资源
2. 根据实际情况在 **各个模块** 中设置相关参数
3. 查看 **顶部总计** 了解可获得的总抽数
4. 通过 **饼状图分析** 了解各模块资源占比
5. 合理规划局长的抽卡策略！
